# SQL Employee Tracker
  

 ## Description
  A command line application that uses data from a SQL database to keep track of employees and which role and department they work in.

  ## Installation
  First you will have to clone the code down from gihub to you machine. Secondly you will have to create the database and tables by running mySQL by using this "mysql -u root -p" in the command line and enter your mySQL password. Once you are on mySQL you can create the database by using "source db/schema.sql" this will create the database and tables. Then enter "source db/seeds.sql" this will populate the database with the seeeded data. From there you can run "npm start" in the terminal wich will start the application.

  ## Usage
  Once fully installed and seeded the database and run "npm start" in the terminal you will be displayed with the questions which you use the arrow keys to navigate and press enter on the option you wish to choose.
